改写自开源项目「findAndReplaceDOMText」，可用正则表达式搜寻任一跨元素节点的文字片段，最终以自定的元素包裹或以其他文字替换目标文本。我将其以ES6改写为模组化的Fibrio，加入Node.js支援，使其可在服务器端使用而毋需依赖DOM API。

Adapted from the open source project findAndReplaceDOMText, Fibrio helps search a certain pattern of text across different text nodes, replace them with something else and/or wrap them in designated elements. Yijun rewrote findAndReplaceDOMText into modular ES6 form, also added support for server-side rendering without the need of DOM API.
